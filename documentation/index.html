Node.js and Yarn

Your computer must have installed nodejs and yarn to run this template. You can download Node.js from https://nodejs.org and yarn from https://yarnpkg.com/lang/en/docs/install/ . NPM comes bundled with Node.js







Install Npm Packages

You must have to install packages. You can do this by running yarn install from the root of your project to install all the necessary dependencies.




Development server

Run yarn start from the root of your project to start a dev server. Navigate to http://localhost:3008/. The app will automatically reload if you change any of the source files.




Build

Run yarn build to build the project. The build artifacts will be stored in the build/ directory.




Eject

Run yarn eject If you are not satisfied with the default configuration. Note: this is a one-way operation. Once you eject, you canâ€™t go back!
